# carnet_d-adresses

# FR
Un carnet d'adresses simple qui permet d'afficher, ajouter, modifier et supprimer des contacts.

Débuté après avoir fini la première partie (P1) du cours gratuit de OpenClassRooms afin de mettre en pratique mes nouvelles connaissances en Python.
J'ai décidé de créer un carnet d'adresses pour m'entraîner à manipuler les listes et les dictionnaires en Python.
Ce projet me permet également de me familiariser avec les boucles les fonctions et les conditions, ainsi que de pratiquer la gestion des entrées utilisateur.


Cours gratuit suivi : (Apprenez les bases du langage Python) https://openclassrooms.com/fr/courses/7168871-apprenez-les-bases-du-langage-python
